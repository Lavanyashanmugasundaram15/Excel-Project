## Project Description
This project is an Excel-based Fitness Tracker designed to monitor daily fitness activities of multiple users.

- Conditional formatting for active and inactive status
- Summary using Pivot Tables and charts

## Excel Skills Used

- Data Validation (Dropdowns)
- Conditional Formatting
- Pivot Tables
- Charts and Dashboard creation

## Outcome
This project helps analyze fitness patterns and track user performance effectively.
